## 문제가 있습니다...

의존성 모듈을 설치하셨나요?
```bash
pip install pyqt5
```

pyqt 경로가 제대로 환경 변수에 추가되었나요?
```
QT_QPA_PLATFORM_PLUGIN_PATH
%userprofile%\AppData\Local\Programs\Python\Python37-32\Lib\site-packages\PyQt5\Qt\plugins\platforms
* 경로는 환경에 따라 달라질 수 있습니다.
```
